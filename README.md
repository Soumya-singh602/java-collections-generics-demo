# Java Collections & Generics Demo

##  Task 4: Collections Framework & Generics

This project demonstrates the use of Java Collections and Generics.

##  Features

- List (ArrayList) for ordered data
- Set (HashSet) for unique elements
- Map (HashMap) for key-value storage
- Generic class DataSorter<T>
- Sorting using Comparator
- Stream API (groupingBy)

##  Classes

- Employee
- DataSorter<T>
- CollectionsDemo (Main class)

##  How to Run

1. Compile:
   javac CollectionsDemo.java

2. Run:
   java CollectionsDemo

##  Concepts Covered

- Collections Framework
- Generics
- Sorting & Searching
- Hashing (equals & hashCode)
- Stream API
